## How this relates to feature-level docs

Product IA (`design/ia/`) governs. Feature IA (`design/features/<slug>/`)
implements. The feature pipeline's UX Flow Agent reads SITEMAP.md and
TAXONOMY.md before proposing new screens or navigation.

## Three IA layers identified (from art direction session, July 2026)

1. **"Where am I?"** — session wayfinding across five acts
2. **"What did I get?"** — brand.md output comprehension and labeling
3. **"What happens next?"** — Stencil → brand.md → Frame transition
